# Wedding Tech Competitor Landscape

## 1. WedMeGood

Website: https://www.wedmegood.com

Headquarters: Gurugram, Haryana, India

Founded: 2014

Founders: Mehak Sagar Shahani and Anand Shahani

Description: One of India’s largest wedding planning platforms connecting couples with vendors like venues, photographers, makeup artists, and decorators.

Key Features:  
• Vendor discovery and reviews  
• Wedding inspiration galleries  
• Budget trackers and checklists  
• Vendor portfolios and pricing

Strengths:  
• Huge vendor marketplace  
• Strong SEO presence  
• Massive inspiration content

Weakness:  
• Planning process still manual  
• Too many listings create decision overload

## 2. WeddingWire India

Website: https://www.weddingwire.in

Description: A global wedding planning marketplace offering tools to organize weddings including vendor discovery, budget tools, and checklists.

Key Features:  
• Vendor directories  
• Budget planning tools  
• Reviews and ratings  
• Wedding checklist tools

Strengths:  
• Global brand credibility  
• Large vendor network

Weakness:  
• Mostly vendor discovery, not automated planning

## 3. WeddingBazaar

Website: https://www.weddingbazaar.com

Description: Indian wedding vendor marketplace where couples can search and hire services like photography, decor, catering, and makeup artists.

Key Features:  
• Vendor search by city  
• Vendor booking platform  
• Wedding inspiration content

Strengths:  
• Large vendor base  
• Strong vendor marketplace

Weakness:  
• Marketplace model only  
• Limited automation in planning

## 4. ShaadiDukaan

Website: https://www.shaadidukaan.com

Description: Wedding vendor marketplace where couples can find verified service providers across categories.

Key Features:  
• Vendor discovery  
• City-based vendor search  
• Reviews and pricing

Strengths:  
• Large vendor catalog  
• Quick vendor comparison

Weakness:  
• No full end-to-end planning system

## 5. Weddingz.in

Website: https://weddingz.in

Description: Platform mainly focused on venue booking and wedding packages.

Key Features:  
• Venue discovery  
• Wedding package deals  
• Venue comparisons

Strengths:  
• Strong venue inventory

Weakness:  
• Narrow focus (venues only)

# Market Insight

Most wedding tech platforms in India operate as vendor discovery marketplaces where couples search, compare, and contact vendors manually.

# Strategic Gap (Opportunity for ShaadiMe)

Current platforms mainly provide vendor directories, inspiration content, and basic planning tools.

Key gap: No instant or automated wedding planning experience.

ShaadiMe positioning: 'Plan your entire wedding in minutes.'

# Competitive Positioning

WedMeGood – Vendor discovery marketplace (Too many options)  
WeddingWire – Planning tools + vendors (Still manual)  
WeddingBazaar – Vendor listing marketplace (No automation)  
ShaadiDukaan – Vendor directory (Limited planning features)  
Weddingz – Venue marketplace (Only venues)  
ShaadiMe – Instant end‑to‑end wedding planning
